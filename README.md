# Nuxt 3 Minimal Starter

Look at the [Nuxt 3 documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

## ติดตั่ง
```bash
ติดตั่ง Node.js, npm https://nodejs.org/en/download
```

เริ่มติดตั้ง โดยใช้ Terminal หรือ Cmd โดยใช้คำสั่งด้านล่างตาม tools ที่ใช้

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## สำหรับ Development

พิมพ์คำสั่งด้านล่าง เพิ่อเริ่มเซิฟเวอร์โดยเข้าผ่าน URL `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## สำหรับ Production

สร้าง Application สำหรับ Production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
